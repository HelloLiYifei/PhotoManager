# 🖥️ 电脑图库 (PhotoManager) - 本地多仓库智能相册与导入管理系统

基于 **Tauri (v2) + Rust + React + SQLite** 开发的极速、轻量本地照片与 RAW 格式管理软件。专门针对摄影师与数码整理爱好者设计，支持物理目录映射、相机存储卡染色导入、多核硬件加速解码、双图对比以及 GPS 轨迹地图。

---

## ✨ 核心特性

### 1. 📁 多工作空间（仓库）系统
*   支持随时新建或切换物理图库仓库（如“生活记录”、“商业拍摄”）。
*   每个工作空间相对独立，配置与 SQLite 数据库均保存在该路径下的隐藏文件夹 `.photomanager/` 中，方便整体移动或备份，具备极高可移植性。
*   顶部提供仓库名直接切换按钮，支持原生文件资源管理器弹窗选择空目录进入。

### 2. 📂 物理目录直接映射与磁盘同步
*   软件中创建的相册在物理磁盘中为直观的同名文件夹：`仓库根目录 -> 相册文件夹 -> 图片`。
*   物理移动照片会同步调用 `fs::rename` 重命名磁盘文件，自动处理目标重名冲突（如 `img_1.jpg`）。
*   提供“同步磁盘相册”功能：启动时或手动同步时，后台会自动扫描磁盘一级子目录，同步入库并智能清除已被用户在外部手动删除的孤立数据库垃圾记录。

### 3. 🎨 染刷式存储卡导入向导
*   支持自定义本地源文件夹或自动侦测插入的相机 SD 存储卡（检测含有 `DCIM` 的可移动磁盘）。
*   **相册染色刷**：点击左侧列表的已有相册（自动哈希计算生成唯一颜色，如红、绿、紫），光标变为画笔，此时点击或按住鼠标左键在右侧照片卡片上**拖拽划过**即可染色分配归属相册。未染色照片默认划归为“默认相册”。
*   **去重防重复导入**：自动根据文件名及文件大小与数据库查重，已导入的照片高亮标记为绿色，附带“已存在”标签并默认取消勾选。
*   **防断电提醒**：导入前温馨提示用户保持设备接触稳定、电脑常开。
*   **稳定网格**：导入页图片尺寸统一，滚动时界面绝不闪烁、抖动或跳格。

### 4. ⚡ 多线程多核硬件加速预览与解码
*   **锁释放与异步并行化设计 (Lock-Free & Async)**：彻底重构照片 Base64 读取逻辑为异步命令 (`async fn`)。锁只占用微秒级，在开始耗时的磁盘读取和图像运算前**瞬间释放数据库连接锁与全局状态锁**，并发读取互不阻塞。
*   **Rayon 硬件加速**：编译链开启 `"rayon"` 特性，将图片旋转、采样等 CPU 运算分配给系统多核并行处理。
*   **Box 采样缩略图优化**：将原本缓慢的插值缩放更换为高性能 Box-sampled 快速采样过滤 (`DynamicImage::thumbnail`)，缩略图清晰度保持 `400px` 超高规格，大图加载速度提升数十倍。
*   **超清预览降级防虚**：支持自动读取 JPEG EXIF 缩略图；若内嵌缩略图尺寸过小（如 `<320px`），后端会自动 fallback 从头解码原图，确保画面边缘及细节绝对平滑。

### 5. ⚖️ 锁定式双图对比模式
*   在瀑布流相册中，选中任意照片点击“对比”按钮，即可将该照片锁定并固定在屏幕右侧大窗展示；
*   左侧区域自动切回画廊滚动列表，用户可以在左侧自由滚动并点击选中其他图片，与右侧的基准锁定图进行同屏像素级细节对照，再次点击即可退出。

### 6. 🌍 照片 GPS 地图轨迹标绘
*   系统自动使用 `kamadak-exif` 解析并读取照片的 GPS 经纬度元数据。
*   集成 Leaflet 地图图层，将有坐标信息的照片标绘于世界地图中，点击标记弹出气泡框，**异步载入该坐标的照片清晰预览图**。
*   对于无网或离线环境，提供优雅的离线地理坐标轨迹卡片列表备用。

### 7. 🗑️ 系统回收站联动
*   在“垃圾桶”分类中，用户可随时“撤销删除”或一键清空垃圾桶。
*   清空时，软件会调用操作系统的原生回收站服务（基于 `trash` 库），**将物理照片移入系统的回收站**而非直接暴力粉碎，提供防误删机制。

### 8. 🏷️ Exif 详细参数与多选标签管理器
*   滑出式右侧属性面板展示照片的详细 Exif 元数据（拍摄时间、相机制造、型号、焦距、光圈、ISO、快门速度、镜头型号等）。
*   提供标签（Tag）卡片栏，支持自由输入回车添加/删除 Tag，并支持在底栏中选中多张照片进行批量打标签。
*   支持在工具栏顶部通过 Tag 下拉菜单筛选照片。

## 🛠️ 技术栈

*   **后端 (Backend)**：`Rust` + `Tauri (v2)` + `tokio` (并发任务) + `rusqlite` (本地库) + `kamadak-exif` (EXIF解析) + `image` (解码与缩略图) + `trash` (系统垃圾箱联动) + `rfd` (原生选择器)。
*   **前端 (Frontend)**：`React` + `Vite` + `CSS` (Glassmorphism 毛玻璃暗黑科技感美学) + `Leaflet` (地图图层)。

## 🚀 快速开始

### 运行环境准备
*   安装 [Rust & Cargo 编译链](https://www.rust-lang.org/tools/install)
*   安装 [Node.js (建议 v18+)](https://nodejs.org/)

### 开发环境启动
1.  克隆本项目并进入根目录：
    ```bash
    git clone <repository_url>
    cd PhotoManager
    ```
2.  安装前端依赖包：
    ```bash
    npm install
    ```
3.  启动 Tauri 桌面端开发调试窗口：
    ```bash
    npm run tauri dev
    ```

### 生产版本打包
若需打包成独立的 Windows 安装程序（如 `.msi` 或绿色版二进制可执行文件）：
```bash
npm run tauri build
```
打包生成的可执行文件将保存在 `src-tauri/target/release/` 下。
